PASO A PASO: Exportar e Importar Dashboard de Grafana (Gauge de Pérdida potencial)

A) EXPORTAR DESDE TU GRAFANA ACTUAL (PC ORIGEN)
1. Abre Grafana en el navegador:
   - URL: http://localhost:3000/
2. Abre el dashboard que contiene el gráfico de velocímetro (Gauge).
   - Menú izquierdo → “Dashboards” → “Browse” → clic en tu dashboard.
3. Exporta el dashboard a JSON:
   - Arriba a la derecha, clic en el ícono “Settings” (engranaje).
   - En el menú lateral de Settings, entra a “JSON model”.
   - Clic en “Download JSON” para guardar el archivo (ej. dashboard_fronteria.json).
   - Nota: el JSON guarda todos los paneles y sus configuraciones (incluye el Gauge y sus umbrales).
4. (Opcional) Anota el nombre del Data Source que usas en tus consultas:
   - En el panel Gauge → “Edit” → arriba “Data source” (nombre, ej. “MySQL Forneria”).
   - Esto te ayudará a mapear el Data Source en el PC destino.

B) INSTALAR E IMPORTAR EN OTRO PC (PC DESTINO)
1. Instala Grafana en el PC destino:
   - Descarga: https://grafana.com/grafana/download (Grafana OSS para Windows, 64-bit MSI).
   - Ejecuta el instalador (siguiente/siguiente hasta finalizar).
2. Arranca Grafana y accede:
   - Abre http://localhost:3000/
   - Usuario/clave por defecto: admin / admin (te pedirá cambiar la clave).
3. Habilitar “embedding” para usar iframes (en el PC destino):
   - Abre el archivo: C:\Program Files\GrafanaLabs\grafana\conf\custom.ini
     (Si no existe, crea “custom.ini” en esa carpeta. No toques defaults.ini.)
   - Agrega estas líneas:
     [server]
     http_addr = 0.0.0.0
     http_port = 3000
     # (Opcional) root_url = http://localhost:3000/

     [security]
     allow_embedding = true

     [auth.anonymous]
     enabled = true
     org_role = Viewer
   - Guarda el archivo y reinicia el servicio “Grafana” en Servicios de Windows.
4. Crea la conexión a tu base de datos (Data Source):
   - Menú izquierdo → “Connections” (o “Configuration”) → “Data sources” → “Add data source”.
   - Elige “MySQL”.
   - Completa:
     - Host: <IP-o-host-de-tu-MySQL> (ej. 127.0.0.1 si MySQL está local en ese PC).
     - Database: forneria (o el nombre real).
     - User/Password: usuario de solo lectura que creaste para Grafana.
   - Clic “Save & test” hasta ver “Database Connection OK”.
5. Importa el dashboard:
   - Menú izquierdo → “Dashboards” → “Import”.
   - Clic “Upload JSON file” y selecciona el archivo que exportaste (dashboard_fronteria.json).
   - En la pantalla de importación:
     - Folder: selecciona o crea una carpeta (ej. “Forneria”).
     - Unique identifier (uid): deja que Grafana genere uno nuevo si no tienes necesidades especiales.
     - Data sources: si te pide mapear fuentes, selecciona el MySQL que creaste en el paso 4.
   - Clic “Import”.
6. Verifica el panel Gauge:
   - Abre el dashboard importado → confirma que ves el velocímetro con tu valor.
   - Si los colores/umbrales no coinciden, revisa el panel:
     - Panel → “Edit” → sección “Field” → “Thresholds”:
       Base (< 100000): rgb(64, 199, 129)
       100000: rgb(255, 214, 0)
       500000: rgb(255, 145, 0)
       1000000: rgb(210, 0, 0)
     - “Standard options”: Min = 0, Max = el tope (ej. 2000000), Unit = currency (o none).
     - “Value options”: Calculate = Last (o Last not null).
   - Clic “Apply” y “Save” el panel si hiciste cambios.

C) INSERTAR EN TU PÁGINA WEB (Django) DEL PC DESTINO
1. Saca el iframe desde Grafana:
   - Abre el dashboard → ubica el panel Gauge → clic “Share” (icono compartir).
   - Pestaña “Embed” → copia el iframe que te entrega (debe apuntar a http://localhost:3000/...).
2. Pega el iframe en tu plantilla de Django (solo en el PC destino):
   - Archivo: templates/dashboard.html
   - Ubica la sección donde quieres el Gauge y pega el iframe, por ejemplo:
     <iframe
       src="http://localhost:3000/d-solo/<UID>/<SLUG>?orgId=1&panelId=<ID>&refresh=30s"
       width="100%" height="250" frameborder="0"></iframe>
   - Guarda y recarga tu dashboard en el navegador.
3. Prueba:
   - Abre tu app Django (http://localhost:8000/dashboard o según ruta).
   - Verifica que el iframe muestra el Gauge. Si aparece login:
     - Confirma que en custom.ini está [auth.anonymous] enabled = true, y que reiniciaste Grafana.
     - Si no quieres acceso anónimo, inicia sesión en Grafana en el mismo navegador antes de abrir tu Django.

NOTAS IMPORTANTES
- Si el MySQL está local en cada PC, usa Host 127.0.0.1 en el Data Source.
- Si el MySQL está en otro equipo, el PC destino debe tener red hacia ese host y credenciales válidas.
- El iframe de cada PC debe apuntar a su propio Grafana local (http://localhost:3000) para que funcione sin depender de un servidor central.
- Al importar el JSON, Grafana puede pedir mapear el Data Source: selecciona el MySQL que configuraste.
- Si luego migras a un servidor central, cambia el src del iframe de “localhost” al IP/dominio del servidor y abre el puerto 3000 en firewall.
